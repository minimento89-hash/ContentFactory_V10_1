Strategia API — Free First

Tutti gli agenti usano gemini-1.5-flash con fallback offline automatico. 15 req/min, 1M token/giorno gratuiti. In caso di 429 il sistema attiva la modalità simulazione senza interruzioni. Configurabile in API Keys Manager.

CIPHER Guardian

Protegge tutto il sistema. Monitor Antigravity, audit file, fix report esportabili, alert sicurezza.

localStorage Bus

Comunicazione inter-agente via cf_chat_messages e disperazione_queue. Zero backend.

Anti-Antigravity

Antigravity sovrascrive file ad ogni sessione. CIPHER rileva le modifiche e genera alert.

Online-Ready

Deploy su GitHub Pages, Netlify o Vercel. Solo HTTPS in produzione. Zero server necessario.
